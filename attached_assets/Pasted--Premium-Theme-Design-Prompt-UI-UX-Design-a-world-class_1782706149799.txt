# Premium Theme Design Prompt (UI/UX)

Design a **world-class, premium, immersive theme system** instead of just changing colors. Every theme should completely transform the application's atmosphere using **animated backgrounds, textures, particles, lighting, depth, smooth transitions, and subtle interactive effects.** The experience should feel alive without affecting readability or performance.

### Global Design Rules

* Every theme must have:

  * Beautiful animated background
  * Smooth 60fps animations
  * Soft glassmorphism where appropriate
  * Elegant shadows and depth
  * Theme-specific icons and accents
  * High readability and accessibility
  * Smooth transitions when switching themes
  * Premium micro-interactions
  * Parallax background movement
  * Subtle ambient particles
  * Dynamic lighting
  * Responsive across desktop and mobile

# Performance Requirements

* Maintain 60 FPS on modern devices.
* Respect `prefers-reduced-motion`.
* Use GPU-accelerated transforms.
* Lazy-load heavy assets.
* Optimize particle systems.
* Ensure animations never reduce readability.
* Preserve excellent contrast and accessibility.

---

# Overall Goal

This should not look like a simple color theme. Each theme must feel like entering a completely different world with cinematic visuals, ambient animations, premium micro-interactions, and immersive storytelling while maintaining a clean, modern, and highly usable user experience.
